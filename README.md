# 📱 Mobile Recharge System (Python)

A simple menu-driven Python program that allows users to select and view different mobile recharge plans.

# 💡 Overview

This project was built as part of my Python learning journey.
It demonstrates how to create a menu-based system using conditional statements and user input.

# ⚙️ Features
* 📋 Menu-driven interface
* 📱 Multiple recharge plans
* 💰 Displays plan details (price, data, validity, benefits)
* ✅ User confirmation system
# 🛠️ Technologies Used
* Python
# 📚 What I Learned
* Using if-else conditions
* Building menu-based programs
* Handling user input
* Structuring simple real-world applications
# 🎯 Purpose

This project was created to practice decision-making logic and simulate a real-world mobile recharge system.
